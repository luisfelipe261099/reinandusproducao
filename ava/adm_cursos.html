Descrição do Curso

Neste curso completo de Desenvolvimento Web Full Stack, os alunos aprenderão todas as etapas necessárias para se tornarem desenvolvedores completos. Desde o front-end com HTML, CSS, JavaScript e React, até o back-end com Node.js, Express e bancos de dados relacionais e não-relacionais.

Os alunos serão capacitados a construir aplicações web completas, responsivas e seguindo as melhores práticas de mercado. Ao final do curso, cada aluno terá um portfólio com pelo menos 3 projetos completos.

O curso inclui mentorias individuais e em grupo, além de acesso à comunidade exclusiva para networking e oportunidades de carreira.
